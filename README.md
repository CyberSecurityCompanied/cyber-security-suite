🔐 Cyber Security Suite

Cyber Security Suite is a modern, desktop-based password security analysis tool built with Python. It provides real-time password evaluation, entropy calculation, crack-time estimation, and a professional cybersecurity dashboard interface.

The project includes a professional installer, dark neon UI design, multi-tab dashboard layout, and features designed to simulate a real-world security analysis environment.

_________________________________________________


🚀 Features

🔎 Real-time password strength analysis

📊 Entropy calculation (security complexity measurement)

⏳ Estimated brute-force crack time

🖥 Modern cybersecurity dashboard interface

📈 Strength history visualization

🔑 Built-in password generator

📝 Activity log system

💾 Exportable security reports

📦 Professional Windows installer support

🌙 Dark neon-themed UI design
